# wasn-style

## Description:
The project hosts an aesthetic and simple LaTeX style suitable for "preprint" publications such as arXiv, techrXhiv and biorXiv, etc. 

## Usage:
1. Use Document class **article**. 
2. Copy **wasn.cls** to the folder containing your tex file.
3. change `\documentclass{article}` to `\documentclass[]{wasn}`.

See **main.tex** 

## Project files:
1. **wasn.cls** - the class file.
2. **template.tex** - a sample template that uses the **wasn style**.
3. **references.bib** - the bibliography source file for paper.tex.
4. **template.pdf** - a sample output of the template file that demonstrated the design provided by the wasn style.
